# opencvjs-chrome-extension-sample
OpenCV.JSをChrome拡張で動かしたサンプルコードになります。

[Qiita記事]()

## これは何？

ポップアップの実行ボタンをクリックするとグレースケール変換するChrome拡張機能(?)です。

![実行前](doc/1.png)
![実行後](doc/2.png)

## 導入方法

1. chrome://extensionsを開く
1. デベロッパーモードON
1. パッケージ化されていない拡張機能を読み込む

`sample.png`は空の画像データなので、任意の画像に置き換えてお試し下さい。
